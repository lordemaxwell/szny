# 正在成为 / Still Becoming

一个为刚毕业、仍在寻找方向的人设计的个人网站。网站有意避免虚构项目与奖项，把“正在成长”本身作为叙事核心。

## 替换个人信息

在 `app/page.js` 中搜索以下文本并替换：

- `你的名字`
- `你的学校 · 你的专业`
- `YOUR_EMAIL@example.com`
- `你的城市`

三个“抽屉”的内容位于文件顶部的 `drawers` 数组，可直接修改。

## 本地运行

```bash
npm install
npm run dev
```

`npm run build` 会生成 Sites 可部署的静态制品目录 `dist/`。
